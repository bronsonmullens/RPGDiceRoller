# RPG Dice Roller
#### A Powerful Dice Roller for D&D and Other TTRPG's

## Features
* Roll custom-sided dice
* Roll up to 100 of each of your custom dice
* Create and save macros (i.e. Fireball 8d6)
* Add modifiers to your macros to make them unique
